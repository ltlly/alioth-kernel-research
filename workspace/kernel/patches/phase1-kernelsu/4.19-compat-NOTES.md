# KernelSU 4.19 Compatibility Patches

## MODULE_IMPORT_NS (added in Linux 5.4)

`drivers/kernelsu/core/init.c` uses `MODULE_IMPORT_NS()` unconditionally for kernels < 6.13.
On 4.19 (< 5.4), this macro doesn't exist. Patched to only emit the import when kernel ≥ 5.4.

```c
#if LINUX_VERSION_CODE >= KERNEL_VERSION(6, 13, 0)
MODULE_IMPORT_NS("VFS_internal_...");
#elif LINUX_VERSION_CODE >= KERNEL_VERSION(5, 4, 0)
MODULE_IMPORT_NS(VFS_internal_...);
#endif
```
